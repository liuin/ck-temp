new PerspectiveCamera(fov, aspect, near, far)

根据 fov, aspect, near, far 生成透视投影相机
Parameters:
Name Type Description
fov float 相机的可视角度,可选参数,如果未指定,初始化为50
aspect float 相机可视范围的长宽比,可选参数,如果未指定,初始化为1
near float 相对于深度剪切面的近的距离，必须为正数,可选参数,如果未指定,初始化为0.1
far float 相对于深度剪切面的远的距离，必须为正数,可选参数,如果未指定,初始化为2000
