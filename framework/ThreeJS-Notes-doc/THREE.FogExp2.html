Class: FogExp2

THREE. FogExp2

指数雾效对象
FogExp2.用来在场景内创建指数雾效,指数雾效是雾效浓度递增根据指数(参数density)设定,Fog对象的功能函数采用
注释内容部分参照 http://blog.csdn.net/omni360

new FogExp2(color, density)

Parameters:
Name Type Description
color THREE.Color 雾效的颜色属性,如果雾效颜色设置成黑色,远处的对象将被渲染成黑色
density float 雾效强度递增指数属性,可选参数,默认是0.00025
